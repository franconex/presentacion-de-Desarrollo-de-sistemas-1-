El Desafío

El Problema que Resolvemos

Una academia de capacitación tecnológica necesita una plataforma web donde ofrecer cursos en línea. Actualmente manejan todo con hojas de cálculo y correos, lo que causa:

⚠ Problemas Actuales

  • Demoras en la activación de suscripciones de forma manual
  • No hay control claro de quién puede acceder a qué curso
  • No existen reportes de ingresos ni cursos populares
  • La experiencia del usuario es mala y genera frustración
  • Errores frecuentes en el seguimiento de estudiantes

Gestión Manual

Hojas de cálculo, correos electrónicos y procesos manuales que generan errores y demoras constantes.

Resultado: Mala experiencia para el usuario final

3 / 15
Nuestra Propuesta

La Solución que Proponemos

Construir un sistema web completo donde los usuarios puedan ver cursos, suscribirse a planes, pagar en línea y acceder al contenido según su plan.

✅ Solución Propuesta

  • Página web pública donde cualquiera puede ver los cursos sin registrarse
  • Sistema de suscripción con 3 planes: Básico, Pro y Premium
  • Pagos automatizados con tarjeta (simulado con Stripe en modo prueba)
  • Login con email, Google o Facebook para facilitar el acceso
  • Panel de administración sencillo para gestionar todo sin saber programar
  • Reportes automáticos de ingresos, cursos populares y estudiantes activos

Plataforma Automatizada

Todo el proceso de inscripción, pago y acceso automatizado con tecnología moderna.

Resultado: Experiencia excelente para el usuario

Proceso de Pago

Flujo de Compra

Proceso completo desde la visualización del curso hasta el acceso al contenido.

1Visitante ve un curso que le interesa
→
2Click en "Suscribirse Ahora"
→
3Si no tiene cuenta → Se le pide registrarse
4Si ya tiene cuenta → Elige plan (Básico, Pro, Premium)
→
5Paga con tarjeta (Stripe modo prueba)
6Pago se confirma → Se activa la suscripción automáticamente
→
7Ya puede acceder al contenido del curso ✅

🔄 Webhook de Stripe: Se utiliza un webhook para procesar el evento checkout.session.completed, asegurando la activación del plan incluso si la conexión del cliente falla.
